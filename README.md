👋 **¡Hola! Bienvenido a mi espacio de trabajo.**

Soy un desarrollador autodidacta que entiende la tecnología no como un conjunto de herramientas neutrales, sino como un reflejo de nuestra forma de pensar y organizar el mundo. En mi día a día estoy gestionando mi propia infraestructura en Debian y windows, asegurando redes y escribiendo código— busco la soberanía técnica. Me interesa construir ecosistemas digitales que respeten la privacidad, eviten las "cajas negras" corporativas y operen con la misma flexibilidad que los sistemas vivos.

Aquí no vas a encontrar soluciones de moda, sino arquitectura pensada, refactorización constante y utilidades creadas para resolver problemas reales desde la raíz.

## 📂 Proyectos Destacados

### ♟️ Simulación de Ajedrez
- **[GM-3000](https://github.com/ElalChico/GM-3000)** – Evolución refactorizada del sistema de autoentrenamiento y simulación de ajedrez.
- **[GM-2000](https://github.com/ElalChico/GM-2000)** – Tablero original de entrenamiento y análisis de partidas.

### 🔐 Seguridad y Cifrado
- **[Silenz-Encriptacion](https://github.com/ElalChico/Silenz-Encriptacion)** – Framework de cifrado AES-GCM 256 bits multi-capa para comunicaciones sensibles.
- **[Silenz](https://github.com/ElalChico/Silenz)** – Interfaz web y documentación del framework Silenz.

### 📱 Transmisión en Tiempo Real
- **[ScreenCutRelay](https://github.com/ElalChico/ScreenCutRelay)** – Transmite pantalla de Android (vía ADB/scrcpy) o webcam a la web en tiempo real como MJPEG sobre HTTP. Incluye panel de control para etiquetas flotantes, recorte de zona y compatibilidad con navegadores, OBS y smart-TVs.
